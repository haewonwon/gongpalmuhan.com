## 프로젝트 개요

clubfair-2026은 동아리 박람회 참여자를 대상으로 성향 매칭과 이벤트 관리를 지원하는 웹 플랫폼입니다. 실시간성이 필요한 사용자 흐름과 운영자가 확인해야 하는 데이터를 함께 다루는 프로젝트입니다.

## 문제 정의

박람회 현장에서는 사용자가 빠르게 참여하고 결과를 확인할 수 있어야 하며, 운영 측에서는 이벤트 진행 상태와 데이터를 안정적으로 관리할 수 있어야 합니다. 프론트엔드는 현장 사용성을 해치지 않는 단순한 흐름과 명확한 상태 표시가 중요합니다.

## 내가 맡은 역할

Frontend와 운영 기여 역할로 참여했습니다. 팀 프로젝트인 만큼 전체 설계와 구현을 단독으로 진행한 것은 아니며, 담당 화면과 운영 과정에서 기여한 부분을 중심으로 상세 내용을 보완할 예정입니다.

## 핵심 구현

- React와 TypeScript 기반 사용자 화면 구성
- Supabase 연동을 고려한 데이터 조회 및 상태 갱신 흐름 정리
- Zustand와 TanStack Query를 활용한 클라이언트/서버 상태 분리
- 운영 상황에서 필요한 화면 피드백과 예외 상태 검토

## 기술 선택 이유

Supabase는 인증, 데이터베이스, 실시간 기능을 빠르게 연결할 수 있어 행사성 플랫폼에 적합합니다. React Query 계열의 서버 상태 관리는 현장 데이터 갱신과 로딩 상태를 분리해서 다루는 데 도움이 됩니다.

## 트러블슈팅

구체적인 문제 해결 사례는 운영 기록과 PR을 확인한 뒤 업데이트합니다. 후보 주제는 Supabase 권한 설정, 동시성, 배포 환경, 현장 트래픽 대응입니다.

## 결과

현재는 프로젝트의 성격과 역할을 설명하는 초안입니다. 확인 가능한 운영 결과와 구현 증거를 기준으로 내용을 확장할 예정입니다.

## 앞으로 개선할 점

- 담당 기능의 화면 흐름과 코드 구조 추가
- 운영 중 발생한 이슈와 해결 과정 정리
- Supabase 보안 정책과 데이터 흐름 설명 보강
