## 프로젝트 개요

mandalart는 만다라트 기법을 활용해 목표와 하위 실행 계획을 구조적으로 관리하는 웹 애플리케이션입니다. 사용자가 목표를 작은 단위로 나누고 진행 상황을 확인할 수 있는 경험을 목표로 합니다.

## 문제 정의

목표 관리 도구는 입력 자체가 복잡해지면 지속적으로 쓰기 어렵습니다. 만다라트의 격자 구조를 웹 UI로 옮길 때는 편집 편의성, 저장 안정성, 모바일 대응을 함께 고려해야 합니다.

## 내가 맡은 역할

개인 프로젝트로 전체 구조를 설계하고 구현했습니다. 다만 이 페이지에서는 실제로 확인 가능한 기능과 코드 범위를 기준으로 계속 보완할 예정입니다.

## 핵심 구현

- Next.js 기반 라우팅과 화면 구성
- Supabase를 활용한 데이터 저장 구조 검토
- Tailwind CSS 기반 목표 카드와 격자 UI 구현 방향 정리
- React Query를 활용한 데이터 요청 상태 관리

## 기술 선택 이유

Next.js는 페이지 단위의 제품 구조를 빠르게 만들 수 있고, Supabase는 초기 제품에서 인증과 데이터 저장을 단순하게 연결할 수 있습니다. Tailwind CSS는 반복되는 UI 조합을 빠르게 구성하는 데 적합합니다.

## 트러블슈팅

구체적인 트러블슈팅은 데이터 모델링, 목표 편집 UX, Supabase 연동 과정에서 실제 해결한 내용을 기준으로 추가합니다.

## 결과

현재는 프로젝트 설명과 케이스 스터디 뼈대를 마련한 상태입니다. 추후 기능 캡처, 데이터 구조, 개선 전후를 추가하면 더 읽기 좋은 상세 페이지가 됩니다.

## 앞으로 개선할 점

- 실제 사용 흐름을 기준으로 화면 캡처 추가
- 목표 데이터 모델과 저장 정책 설명
- 편집 UX에서 고민한 지점과 개선 결과 정리
