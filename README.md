# 공팔무한

안녕하세요. `김해원`입니다.

`공팔무한`은 **무에서 무한으로 향하는 저의 여정을 담은 공간**입니다.  
`김해원`이라는 사람을 잘 보여줄 수 있도록 이것저것 많이 시도해 보겠습니다.

## HOW TO RUN

```bash
pnpm install
pnpm dev
```

## TECH

- Astro
- React (islands)
- Tailwind CSS

## STRUCTURE (Astro)

```text
src/
  components/
  layouts/
  pages/
  styles/
public/
```
